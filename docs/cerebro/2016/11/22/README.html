Introduction

Cerebro is a web dashboard to subscribe, check and manage notifications over Graphite data.

Please check the Cerebro wiki.

What is included?

  • services : REST webservices over backends, such as Seyren, writen in Spring/Java

  • dashboard : user interface, writen in Angular 2 (TypeScript + Bootstrap)
